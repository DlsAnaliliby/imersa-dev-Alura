Este projeto foi desenvolvido durante a Imersão Dev da Alura, com o objetivo de criar um site simples e interativo que funciona como uma base de conhecimentos sobre linguagens de programação.

A aplicação apresenta uma lista de linguagens de forma clara e organizada, permitindo que o usuário explore informações essenciais sobre cada uma. A ideia do projeto é treinar conceitos fundamentais de desenvolvimento web, como estruturação de páginas, estilização e manipulação do DOM com JavaScript.

 - Funcionalidades

Exibição de diferentes linguagens de programação

Estrutura organizada em cards

Conteúdo simples e direto

Interface leve e responsiva

 - Tecnologias Utilizadas

HTML5 – estrutura do site

CSS3 – estilização e layout

JavaScript – lógica e interação da página

- Objetivo do Projeto

Praticar os conceitos aprendidos durante a imersão, reforçando lógica de programação, manipulação de dados e desenvolvimento de interfaces web. Além disso, criar um projeto que possa servir como consulta rápida para iniciantes que estão conhecendo novas linguagens.
